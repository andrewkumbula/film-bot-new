# Развилка при подборе фильмов

## Идея

После нажатия «Подобрать фильм» пользователь сначала выбирает **источник/режим** подбора, затем идёт по соответствующей ветке.

## Варианты (кнопки первого шага)

| Вариант | Описание | Данные |
|--------|----------|--------|
| **Оскар** | Номинанты и победители премии «Оскар» | Список фильмов по категориям (лучший фильм, режиссура, сценарий и т.д.). Нужен источник: API или статичный JSON/БД. |
| **Кинопоиск Топ 250** | Фильмы из рейтинга Кинопоиска | Уже есть API (poiskkino.dev) — можно запросить топ или использовать готовый список. |
| **Кинофестивали** (позже) | Канны, Венеция, Берлинале, «Кинотавр» и др. | Отдельные списки по фестивалям/годам. |
| **Обычный подбор** | Текущий флоу | Настроение → жанры → длительность → возраст → с кем смотрим → чего избегать → ИИ даёт 5 фильмов. |

## Схема флоу

```
[Подобрать фильм]
        │
        ▼
┌───────────────────┐
│ Выбери режим:     │
│ • Оскар           │
│ • КП Топ 250      │
│ • Кинофестивали*  │
│ • Обычный подбор  │
└───────────────────┘
        │
        ├── Оскар ──────────► [уточнения по году/категории?] ─► выбор из списка/ИИ по списку
        │
        ├── КП Топ 250 ─────► [жанр/год опционально?] ─► фильтрация топа, выдача 5
        │
        ├── Кинофестивали ──► (позже) выбор фестиваля → год → выдача
        │
        └── Обычный подбор ─► настроение → жанры → длительность → возраст → компания → чего избегать → ИИ 5 фильмов
```

## Реализация по шагам

1. **Развилка в коде**  
   Новый первый шаг (state `source`): клавиатура с 4 кнопками. В `preferences` сохраняем `source`: `oscar` | `top250` | `festivals` | `default`.

2. **Обычный подбор**  
   При `source=default` переходим в текущий state `mood` и дальше без изменений.

3. **Оскар**  
   - Нужен источник данных (номинанты/победители по годам и категориям).  
   - Варианты: ручной JSON, парсинг сайта, готовый API.  
   - Флоу: выбор года (и опционально категории) → фильтрация списка → можно отдать ИИ «выбери 5 из этого списка по настроению» или просто показать топ по рейтингу.

4. **Кинопоиск Топ 250**  
   - У poiskkino.dev есть лимиты; топ-250 обычно отдают отдельным методом или списком.  
   - Флоу: опционально жанр/год → фильтруем топ → выдаём 5 (случайно или по рейтингу/настроению через ИИ).

5. **Кинофестивали**  
   - Отложено: нужны списки по фестивалям и годам, затем аналогичная логика выдачи.

## Логирование (flow_log)

Для развилки в лог пишем шаг `source` и значение `oscar` / `top250` / `festivals` / `default`, затем шаги выбранной ветки как сейчас.

## Итог

- Сначала делаем **развилку в боте** и ветку **«Обычный подбор»** (текущий флоу).  
- Ветки **Оскар** и **КП Топ 250** можно сначала показать как «скоро» или заглушку, затем подключить данные и логику выдачи.
